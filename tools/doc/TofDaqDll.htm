TofDaqDll.dll version 1.99

The functions contained in TofDaqDll.dll are used to communicate with the TofDaq recorder application. They are used to configure the recorder application, control running acquisitions, give access to data being recorded and allow for arbitrary data storage synchronous and asynchronous to the recorded mass spectra.


Control functions

This section describes functions that are used to manage DLL internals (initialize, clean up, version number), status checking of the recorder application (running, active) and actual acquisition control (start, stop, continue).

Configuration functions

The API provides different ways to configure the TofDaq recorder application. All configuration options for a given data acquisition hardware/timing mode combination can be configured interactively with the graphical configuration dialogs built into the TofDaq recorder application. These dialogs are rather complicated due to their completeness and therefore the programmer creating a custom GUI based on this API will often choose not to confront the user with these dialogs.

A configuration file (*.ini) holds a complete configuration and one of the easiest ways to change TofDaq settings is to load complete configurations.

For more fine-grained access to the configuration single parameters can be changed.
